# Challenges

> **Active impediments — what is in the way right now.** Challenges are *team-level* or *system-level* — not the day's tactical blockers (those live in `memory/current-focus.md`). A challenge stays here until it is either resolved (move to retired) or accepted as a permanent constraint (move to `memory/effectus/beliefs.md` as a stated reality).

## How to write a challenge

- **Name what it blocks.** Which goal in `memory/effectus/goals.md` does this impede?
- **Name what would unblock it.** What concrete change in the world would close this challenge?
- **Date-stamp.** When did this challenge become visible? Long-standing challenges deserve different treatment than fresh ones.
- **Status.** `open` / `partial` / `resolved` / `accepted-as-constraint`.
