# Goals

> **Active, dated commitments.** What the team has agreed to deliver in the current quarter or year. Each goal is concrete, has a deadline, and rolls up to a mission in `memory/effectus/mission.md`. Goals change quarterly.

## How to write a goal

- **Concrete.** "Ship X by Y" or "Reduce metric Z below threshold W by date D."
- **Dated.** Every goal has a target date. "Soon" is not a date.
- **Rolled up.** Every goal names which mission (M1/M2/M3) it serves.
- **Falsifiable on the target date.** On the deadline, anyone reading this file can say "shipped" or "missed".
