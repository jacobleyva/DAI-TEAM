# Mission

> **The long-arc purpose of this team.** What we are for. Why we exist as a group, beyond the current quarter's projects. Mission changes on the year-or-more cadence — if it changes monthly, it is a goal, not a mission.

## How to write a mission statement

- One to three sentences per mission.
- Name what the team makes possible *for someone outside the team*.
- Avoid vague verbs ("enable", "empower", "transform"). Use specific verbs ("ship", "operate", "audit", "design", "rebuild").
- A good mission can be falsified. If every team can claim it, it is too vague.
