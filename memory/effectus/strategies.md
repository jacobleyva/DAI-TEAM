# Strategies

> **How we intend to pursue each goal.** Strategies are the route, not the destination. Each strategy names which goal it serves, which beliefs it operates from, and which models it relies on. A strategy is revisable when its underlying belief or model changes.

## How to write a strategy

- **Tied to a goal.** Strategy without a goal is opinion. State the goal ID it serves.
- **Names the leverage.** What unfair advantage or specific approach makes this strategy work?
- **Names the trade-off.** What does this strategy give up? Strategies with no trade-off are usually wishful.
- **Revisable.** A strategy is not a commitment — it is a current best route. When the route stops working, change the strategy without changing the goal.
