# Wisdom

> **Hard-won lessons, written down so they don't need to be re-earned.** Each entry is one sentence — operate-able guidance that came out of real pain. If a lesson cannot be compressed to a sentence, it is two lessons. If it has no past pain behind it, it is opinion, not wisdom.

## How to write a wisdom entry

- **One sentence.** Compress until it fits.
- **Past-tense origin in parens.** Brief — what burned us, dated. So a future reader knows the lesson is real, not theoretical.
- **Imperative.** "We do X" / "We never Y" — not "It's good to consider Z."
- **Append-only.** Wisdom doesn't get deleted. If a lesson stops applying, mark it with a reason and a date — don't remove it.
